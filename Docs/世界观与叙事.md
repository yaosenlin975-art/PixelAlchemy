## 0. 与 ADR 的边界声明（强制）

- 本文件所有内容均为「文案 / 主题命名 / 氛围」，**不绑定任何玩法机制变更**。
- 严格遵循 ADR §0 五点裁决：**Kill 默认 / 不缩圈 / 留 Dash（无 i-frames）/ 无 HP / 固定力度 + 循环切换**。
- 死亡唯一定义＝**出界即死（坠入虚空）**；无任何 HP / 血量 / 第二死因叙事。
- 环境材料（火 / 毒 / 酸 / 熔岩）叙事定位＝「逼走位、控场」的工具，**非致死**；唯一真正的死因是虚空。
- 本作为竞技对战，叙事仅作包装，**不引入剧情流程 / 章节 / 过场**。

## 1. 一句话命题 + 竞技场背景

**一句话命题**：

> 在被废弃的造神工坊之上，悬浮着无主的竞技场「虚渊」；一群由「造神者」留下的器灵以魔法彼此角力，败者越过边界、坠入虚空、归于无——而虚空之下，再无复活。

**竞技场背景（为何众人于此以魔法相搏、败者坠入虚空）**：

上古的「造神者」曾试图以魔法与材料捏塑出完美的神，却在一次「逆造」中尽数陨落，只留下悬于虚空的工坊残骸。他们造出的「器灵」（被赋予法术之骨的造物）未被赋予神位，仍在残墟中彼此试炼，执念只有一个：**证明自己不是「器」，而是配得为「神」的那一个**。竞技场的边界外即是「虚渊」——造神失败的产物尽数沉没之处；任何器灵一旦越过边界，便被虚空吞没、再无归途。于是每一场对决，都是一次「谁先坠渊、谁独存」的永恒试炼。

**基调来源（用户偏好参考，非游戏内剧情）**：神学工程师、人造神、命运操纵、悲剧救赎——以《逆造之神》主角「沈不器」（君子不器，不甘为器）为情绪锚点，但本作仅取其「挣脱被造宿命」的悲剧张力作为氛围，不做故事化展开。

## 2. 核心隐喻与基调（机制 → 叙事映射）

| 机制（ADR 裁定） | 叙事隐喻 | 文案基调 |
|---|---|---|
| 出界即死（D1 / D4） | 坠入虚渊 | 死亡不是「被打败」，而是「被世界抛弃」；冰冷的虚无感 |
| 无 HP（D4） | 一坠即灭 | 没有挣扎余地，败即归无 |
| 环境材料非致死 | 材料是「逼你走向边界」的手 | 火 / 水 / 毒 / 酸是推手，不是杀手 |
| 击退推出界（K 表） | 「推堕」 | 杀招本质是让对手自己踏空 |
| Dash 无 i-frames（D3） | 纵能疾驰，亦难逃坠 | 即使有位移，仍会被推下——呼应 D4 禁免推 |
| 固定力度（D5） | 法术之骨既定 | 不靠蓄力，只靠时机与位置 |
| 最后存活者胜 | 独存的器灵 | 胜者成为本场唯一的「存神」 |

## 3. 阵营 / 流派（玩家可感知的主题命名，中文，纯包装）

> **说明**：以下「流派」仅作为**可选的主题旗帜 / 名片**（自建房或排位名片展示），**不影响任何数值、法术、能力**。玩家可自由署名任一流派，表达自身法术偏好与审美。命名倾向中文，呼应材料与法术主题。

| 潮生盟 | 水青 `0x029F` | 「潮律所至，万物退避。」 | Water / Ice |

## 5. 玩家身份 / 角色基调（造物命名，纯氛围）

- 玩家角色统称：**器灵**（被造神者赋予法术之骨的造物）。
- 胜利者叙事称谓：**存神**（本场唯一未被虚空吞没者）。
- 败者叙事称谓：**坠渊者**（已归于虚渊）。
- 自建房房主可署名「主造」；排位禁对手文本（GDD §4.1），故身份呈现仅本地可见。
- 取名倾向：中文单字 / 双字雅称（如「不器」「凝滞」「潮生」），呼应《逆造之神》「不甘为器」的母题；**不强制，玩家可自定**。

## 6. 结算叙事（名次 / 击杀 的叙事呈现；引 结算界面规格.md）

> 本节约定结算界面的**文案层包装**，所有数据仍来自权威 `ResultPayload`（结算界面规格 §3.3），客户端只展示不重算。状态色严格沿用 结算界面规格 §6.2（胜＝冰青 `0x73FF` / 负＝火橙 `0xF900` / 平＝灰 `0xA294`）。

| 结算态 | 状态色（引 §6.2） | 叙事横幅文案 | 含义 |
|---|---|---|---|
| 最后存活者胜（Kill 收束） | 冰青 ★ | 「虚渊独存 · 你成为本场唯一的存神」 | 你是唯一未坠者 |
| 败（出界） | 火橙 ✗ | 「你坠入了虚渊。」 | 你被推下边界 |
| 平局（Bounce / Wrap 超时 ≥2 存活） | 灰 ＝ | 「虚渊未决 · 无存神诞生」 | 无人独存 |
| 全员出界 | 火橙 ✗ | 「尽数归无 · 最迟坠渊者居首」 | 按出局序倒推 |

**名次榜叙事包装**（引 结算界面规格 §2.3 按出局顺序倒推）：

- 榜单标题：「虚渊名录」（排名榜）。
- 每行呈现：名次 + 流派旗帜（可选）+ 击坠数 K + 存活时长。
- 击坠数 K 叙事化：「击坠 K」＝你将 K 名对手推入了虚渊（击杀归属按 GDD §11 Q3「环境 / 生物致死按最后接触玩家归属」，待拍板后接入）。
- 本地玩家行：冰青描边高亮（引 §6.2），文案标注「（你）」。
- 平局：共享名次处显示「＝」与中性灰，不展示胜负横幅（引 §2.4）。

**P1 复盘指标叙事化**（引 §3.2 P1 项，可选展开）：

- 施法统计 → 「术式施用 N 次」
- 材料改造量 → 「改写虚渊地貌 N 像素」

## 7. 边界模式叙事变体（呼应 GDD §4.2 / ADR D1）

| 边界模式 | 叙事包装 | 备注 |
|---|---|---|
| `Kill`（默认） | 虚渊无底，越界即坠 | 核心体验，唯一标准竞技 |
| `Bounce`（自建房） | 残墟壁垒反弹，边界不可杀 | 娱乐变体，材料战为主 |
| `Wrap`（自建房） | 虚渊环抱，对侧重现 | 混沌娱乐局 |

> 仅 `Kill` 为竞技标准；Bounce / Wrap 叙事上解释为「造神者遗留的不同残墟规则」，不进排位。

## 8. 明确边界：纯包装声明（不引入剧情流程）

- 本作是**竞技对战**，世界观仅作**表现层包装**：无主线、无章节、无过场动画、无角色剧情对话树。
- 所有叙事元素（流派旗帜、材料 / 法术叙事名、结算文案、身份称谓）**均为可选展示文本**，**零机制绑定**。
- 不与 ADR 任何裁决冲突：死亡＝出界即死（坠渊）、无 HP、无缩圈、Dash 无 i-frames、固定力度、循环切换。
- 任何「故事」均可通过后续 P1 内容以**非阻塞**方式叠加（见 §9），不影响对局节奏与确定性。

## 9. 后续可扩展方向（P1，非阻塞，纯表现层）

1. **流派名片系统**：自建房 / 名片展示玩家署名流派 + 信条 + 旗帜色，纯展示，不影响匹配与数值。
2. **造神档案（Meta 风味）**：账号级「器灵谱」收集（解锁叙事称号 / 旗帜皮肤），无数值增益，依赖运营 / 商业化文档（GDD §9 暂缓）。
3. **主题竞技场皮肤**：不同「虚渊层」视觉皮肤（深渊 / 残坊 / 星墟），仅换 ART 调色板取色（须守 RGB565 约束），不改地形规则。
4. **击坠 / 坠渊 快捷表情与语音 flavor**：在既有的 QuickChat 通道（GDD §4.2 观战表情、UX §2.2）追加主题化文案（如「归墟」「坠」），不新增通道。
5. **结算战绩卡主题化**（引 结算界面规格 §5.5）：战绩卡加入流派旗帜与水印，仍守 RGB565 / 像素字体约束。

## 10. 文案风格指南（语气 / 术语 / 本地化注意）

- **语气**：冷峻、克制、带悲剧余韵；避免搞笑 / 卖萌 / 网络梗，保持「造神试炼」的庄重感。
- **术语表（中英对照，便于本地化）**：
  - 虚渊 = the Void / the Abyss（出界即死的虚空）
  - 器灵 = the Made / the Vessel-spirit（玩家角色）
  - 存神 = the Surviving / the Last Remaining（胜者）
  - 坠渊者 = the Fallen（败者）
  - 流派 = School / Faction（主题旗帜）
- **本地化注意**：命名以「专有名词 + 可直译短句」结构，避免中文谐音梗、成语双关、文化硬梗；法术叙事名采用「名词 + 术 / 弹 / 壁」等可拆解构词，便于英日韩本地化时保留结构。所有文案字符串集中，便于提取。
- **与 ART / UX 一致**：状态色严格引用 结算界面规格 §6.2 三色；不在叙事文本中暗示任何板外色或新机制。
